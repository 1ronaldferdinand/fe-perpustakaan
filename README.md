## Installation

### Prerequisites

- Node.js and npm installed on your machine
- Node.js => 18.13.0^
- NPM => 8.19.3^

#### Install Dependencies
```
npm install
```

#### Run the Vue development server
```
npm run dev
```
- The Vue development server by default will start at http://localhost:5173.
